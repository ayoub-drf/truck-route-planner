#### **Installation & Setup**  
```bash
git clone https://github.com/ayoub-drf/truck-route-planner.git
cd truck-route-planner
cd src/frontend/ && npm install   # For frontend dependencies

```

#### **Run the Project**  
```bash

# Start Frontend (React or Vite)
```
